Ionic CouchDB Base
=====================

Getting Started
--------------

* git clone https://github.com/jrhicks/ionic-pouchdb.git

* cd ionic-pouchdb-base

* npm install

* bower install

* gulp

* http://localhost:9000

CouchDB 1.5 VM
---------------

* cd couchdb_vm

* vagrant up

* http://localhost:5984/_utils/
